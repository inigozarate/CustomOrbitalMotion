## INSTALLATION
Go to your Foam Folder:

cd ../$FOAM_RUN

Save the folder with name custom_orbitalmotion and enter inside it:

cd /CustomOrbitalMotion

Go inside the first folder:

cd mrigidBodyDynamics

Compile it as a library

wmake libso

Do the same for the second folder:

cd ../mrigidBodyMeshMotion

Compile it as a library:

wmake libso
